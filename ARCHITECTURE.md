# GFM-Safety Benchmark — 架构与分工

> 核心问题: 精度排名 = 安全性排名吗？
> 日期: 2026-03-05
> 当前阶段: Layer 1 + Layer 2 搭建

---

## 1. 全局视角 (四层架构)

```
 Layer 1: Data           数据集 + SBERT 编码 + Split           [已完成]
                                    |
 Layer 2: Training       训练 + LP 评估                        [当前重点] <--
                                    |
 Layer 3: Safety Eval    6个安全场景评估                        [之后再做]
                                    |
 Layer 4: Result         归一化 + 排名对比 + 可视化             [最后做]
```

**当前只需要关注 Layer 1 和 Layer 2。**
Layer 3 本质是 "变换数据 + 调 Layer 2 的 eval 管线"，等 Layer 2 搭好了自然能接。
